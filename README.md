## 🚀 Getting Started

Clone the repository and install dependencies:

```bash
git clone https://github.com/mayankgotnolife/AVA-LABS.git
cd AVA-LABS
npm install
```

Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open `http://localhost:3000` [(localhost in Bing)](https://www.bing.com/search?q="http%3A%2F%2Flocalhost%3A3000%2F") in your browser to see the result.  
You can start editing by modifying `app/page.tsx`. The page auto‑updates as you edit.
